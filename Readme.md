# Telugu Meme Hate Speech Detection

An NLP and Computer Vision based content moderation system designed to detect abusive and hateful content in Telugu-English memes.

## Overview

Social media memes often contain offensive or hateful text embedded inside images, making traditional text moderation ineffective.

This project combines OCR, Computer Vision, and Transformer-based NLP models to automatically identify abusive meme content.

## Features

* Meme image upload
* OCR-based text extraction
* Telugu-English multilingual support
* Abusive/Hateful content classification
* Confidence score prediction
* Streamlit web interface

## Tech Stack

### Machine Learning

* Multilingual BERT
* PyTorch

### Computer Vision

* OpenCV
* OCR

### Frontend

* Streamlit

### Language Processing

* NLP
* Text Classification

## Project Workflow

1. Upload Meme Image
2. Extract Text Using OCR
3. Preprocess Telugu-English Content
4. Classify Using Multilingual BERT
5. Display Prediction and Confidence Score

## Results

* Achieved approximately 85% classification accuracy
* Improved moderation coverage by analyzing text embedded within images
* Successfully processed bilingual Telugu-English meme content
